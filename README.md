# Resistor-color-code-checker

This is a C++ code that checks the value of a given resistor

## Install any C++ Compiler

```bash

cd /path/to/destination

git clone https://github.com/AustinChris1/Resistor-color-code-checker.git

cd Resistor_color_code_checker

g++ Resistor_color_code_checker -o 

./Resistor_color_code_checker

```
